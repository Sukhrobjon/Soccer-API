# Soccer API

### Available Endpoints
```javascript
/leagues
/leagues/:leagueId
/leagues/:leagueId/teams
/leagues/:leagueId/teams/:teamsId
/leagues/:leagueId/teams/:teamsId/matches
/leagues/:leagueId/teams/:teamsId/players
/leagues/:leagueId/teams/:teamsId/players/:playersId
```
